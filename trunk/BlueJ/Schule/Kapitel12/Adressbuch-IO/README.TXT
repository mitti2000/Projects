Projekt: Adressbuch-IO
Autoren: David J. Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 5. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2012

Eine einfache Implementierung eines Adressbuches.

Startpunkt: 
    Entweder: Sie erzeugen ein Objekt der Klasse Adressbuch
    und fügen Kontaktdaten (wie Name, Adresse, Telefon) ein.
    Oder: Sie erzeugen eine AdressbuchDemo und lassen sich
    das Adressbuch mit den Testdaten geben.
    Erzeugen Sie dann einen AdressbuchDateiverwalter, mit
    dem einige Ein-/Ausgabe-Operationen auf einem Adressbuch
    demonstriert werden.

Benutzung des Verwalters:
    + Suchergebnisse werden in der Datei "Ergebnisse.txt"
      im Projektordner gespeichert.
    + Die Datei "Beispiel.txt" enthält die Daten eines Kontaktes,
      die dem Adressbuch mit der Methode 'neueKontakteAusDatei'
      des Verwalters hinzugefügt werden können.
    + Ein komplettes Adressbuch kann in binärer Form gespeichert
      werden (Methode 'speichereInDatei'). Dieses kann wieder 
      eingelesen werden mit der Methode 'liesAusDatei'.
