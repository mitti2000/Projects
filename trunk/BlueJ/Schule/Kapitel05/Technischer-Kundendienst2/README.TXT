Projekt: Technischer-Kundendienst2
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 5. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2012

Es wird in Kapitel 5 diskutiert.

Dieses Projekt implementiert eine zweite Version des Kundendienstsystems.
Es ist eine Zwischenversion, um die ersten Änderungen zu illustrieren, die
in diesem Kapitel diskutiert werden.

Um das System zu starten, erzeugen Sie ein Exemplar der Klasse
Kundendienstsystem und rufen die Operation "starten" auf.

Geben Sie dann eine Beschreibung Ihres Problems in der Konsole ein.
