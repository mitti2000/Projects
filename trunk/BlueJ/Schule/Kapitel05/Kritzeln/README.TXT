Projekt: Kritzeln    
Autoren: Michael Kölling und David J. Barnes

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 5. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2012

Ziel: Demonstration der grafischen Möglichkeiten auf einer Canvas-Klasse.

Dieses Projekt bietet einige kurze grafische Demonstrationen in der
Klasse BallDemo. 

Dieses Projekt bietet viel Raum für Erweiterungen.
Die Klasse Canvas braucht üblicherweise nicht geändert zu werden.
Sie sollte eher wie ein Bibliotheksklasse behandelt werden (öffnen
Sie den Editor und wechseln Sie in die Schnittstellensicht). 

Die anderen Klassen können geändert, erweitert oder um weitere Klassen
ergänzt werden. Das Buch gibt Anregungen für verschiedene Übungen.
