Projekt: Weblog-Auswertung
Autoren: David Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 5. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2012

Zweck des Projektes: Die Benutzung von Arrays illustrieren.

Startpunkt: Erzeugen Sie ein Exemplar von ProtokollAuswerter.

Der LogdateiLeser versucht eine Datei zu lesen (weblog.txt),
die in jeder Zeile folgende Daten enthalten soll:

    Jahr Monat Tag Stunde Minute
    
Monate sollten als Werte von 1 bis 12, Tage als Werte von 1 bis 31
angegeben sein. Falls die Beispieldatei nicht gefunden werden
kann, generiert der Leser Simulationsdaten. Sie können aber auch mit
LogdateiErzeuger zufällige Daten erzeugen. Verwenden Sie dazu die 
erzeugeDatei-Methode und übergeben Sie einen Dateinamen und die Anzahl 
der zu erzeugenden Einträge.
